# 網頁抽點器

給老師課堂使用的隨機抽點工具。可直接用瀏覽器開啟 `index.html`，不需要安裝任何套件。

## 功能

- 中間顯示目前隨機跳出的學生姓名。
- 可上傳 CSV 產生名單，支援 `座號` 與 `姓名` 欄位。
- 若 CSV 有 `班級` 欄位，頁面會自動顯示成該班級的抽點器。
- 「開始」按鈕開始快速輪播姓名。
- 「停止」按鈕停止並保留抽到的姓名。
- 可在頁面上編輯學生名單，每行一位學生。
- 名單會儲存在瀏覽器 `localStorage`，下次開啟同一瀏覽器仍會保留。
- 公開版不內建真實班級與學生名單。使用時可在頁面中自行貼上名單，資料只會存在目前瀏覽器的新版 `localStorage` key。
- CSV 中若有帳號、電子郵件等欄位，程式不會匯入那些欄位。
- 舊版本機名單 key 會在載入時清除，避免瀏覽器帶回先前測試用名單。

## 開啟方式

直接用瀏覽器開啟 `index.html`。
