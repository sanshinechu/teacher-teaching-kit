# 運算思維流程畫布

## 目的

建立一個免登入、可直接開啟的極簡流程圖白板，讓學生在進入實體程式積木拼貼前，先用流程圖整理專案邏輯；也方便老師在台上快速解構遊戲機制。

## 使用對象

- 年級：國小高年級以上
- 科目：資訊、Scratch、專題課程
- 使用者：教師 / 學生

## 專案類型

- 靜態網頁工具

## 結構

```text
運算思維流程畫布/
├─ README.md
├─ index.html
├─ styles.css
└─ script.js
```

## 使用方式

1. 開啟 `index.html`。
2. 使用工具列新增「開始」、「執行」、「判斷」、「輸入」、「輸出」。
3. 拖曳圖形調整位置。
4. 點兩下圖形修改文字。
5. 開啟「連線模式」，依序點兩個圖形建立箭頭。
6. 完成後可下載 PNG 留作課堂紀錄。
