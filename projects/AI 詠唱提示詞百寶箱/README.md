# AI 詠唱提示詞百寶箱

## 目的

建立一個可快速複製、修改與組合生成式 AI 指令的課堂提示詞面板，降低學生第一次使用 AI 影像生成、劇本創作或學習輔助工具時的門檻。

## 使用對象

- 年級：國小高年級以上
- 科目：資訊、藝術、語文、專題課程
- 使用者：教師 / 學生

## 專案類型

- 靜態網頁工具

## 結構

```text
AI 詠唱提示詞百寶箱/
├─ README.md
├─ index.html
├─ styles.css
└─ script.js
```

## 使用方式

1. 開啟 `index.html`。
2. 依課堂任務選擇「影像生成」、「劇本創作」或「學習任務」。
3. 點擊卡片可把基礎提示詞放到右側編輯區。
4. 修改 `{變數}` 內容後，按「複製修改版」貼到生成式 AI 工具。
