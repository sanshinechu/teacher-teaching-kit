# 教師用教學套件

這個資料夾是多個教學小專案的 workspace。根目錄負責放共用說明、模板、素材與專案索引；每個實際工具或教材包請放在 `projects/` 底下，各自維持獨立結構。

## 目錄規劃

- `projects/`：主要小專案，每個子資料夾是一個獨立專案。
- `_templates/`：可重複使用的專案、課程、教材模板。
- `_assets/`：跨專案共用素材，例如圖片、圖示、音檔、字型授權紀錄。
- `_docs/`：workspace 層級文件，例如專案索引、規格、維護紀錄。
- `archive/`：已完成、暫停或不再維護的小專案。

## 新增小專案

1. 在 `projects/` 內建立新資料夾。
2. 複製 `_templates/project_README.md` 到新專案內，改名為 `README.md`。
3. 在 `_docs/project-index.md` 加上一筆紀錄。
4. 依專案類型建立自己的檔案，例如網站、Python 工具、簡報、講義或素材包。

## 建議分類

- 教材文件包：適合講義、學習單、課程設計。
- 靜態網頁工具：適合可直接開啟的互動教材。
- React/Vite 工具：適合較完整的互動網站或教學系統。
- Python 工具：適合批次產生題目、整理資料、轉檔、自動化。
